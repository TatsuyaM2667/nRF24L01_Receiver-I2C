# nRF24L01_Receiver-I2C
このコードはnRF24L01のデータを受信し、I2C経由で送信するコードです。

| ESP32      | nRF24L01　  |
|:-----------|------------:|
|   3.3V     |  VCC        | 
|   GND      |  GND        | 
|   CE       |  GPIO17     | 
|   CSN      |  GPIO5      | 
|   SCK      |  GPIO18     | 
|   MOSI     |  GPIO23     | 
|   MISO     |  GPIO19     |
